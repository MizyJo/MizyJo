Aide

Fonctionnalités du Dashboard

  • Consultation de tous les contenus du site :

    Vous pouvez consulter la liste de tous les contenus présents sur le site web.

  • Modification des contenus :

    Vous pouvez modifier les contenus du site web, y compris ceux accessibles via le menu "Autre modification du site

  • Suppression des contenus :

    Vous pouvez supprimer les contenus du site web, y compris ceux accessibles via le menu "Autre modification du site

Ajout de contenus

  • Ajout de services :

    Si vous êtes autorisé à ajouter des services, le bouton correspondant apparaîtra dans le menu. Cette page vous permet d'ajouter de nouveaux services.

  • Ajout de départements :

    Si vous êtes autorisé à ajouter des départements, le bouton correspondant apparaîtra dans le menu. Cette page vous permet d'ajouter de nouveaux départements.

  • Ajout d'actualités :

    Si vous êtes autorisé à ajouter des actualités, le bouton correspondant apparaîtra dans le menu. Cette page vous permet d'ajouter de nouvelles actualités.

  • Ajout de prix :

    Si vous êtes autorisé à ajouter des prix, le bouton correspondant apparaîtra dans le menu. Cette page vous permet d'ajouter de nouveaux prix.

  • Ajout d'informations "A propos" :

    Si vous êtes autorisé à modifier les informations "A propos", le bouton correspondant apparaîtra dans le menu. Cette page vous permet de modifier le contenu de la page "A propos".

  • Autres modifications du site :

    Cette page regroupe les contenus du site non mentionnés ci-dessus. Vous pouvez modifier divers éléments comme le texte d'accueil du site. Seuls les utilisateurs autorisés peuvent accéder à cette page.

Gestion des utilisateurs

  • Ajout d'un nouvel utilisateur :

    Si vous êtes autorisé à ajouter des utilisateurs, le bouton correspondant apparaîtra dans le menu. Cette page vous permet d'ajouter de nouveaux utilisateurs avec un groupe d'utilisateurs par défaut.

  • Gestion des utilisateurs :

    Vous pouvez supprimer ou désactiver les utilisateurs dans ce bouton sur le menu.

Historique de connexion

  • Consultation de l'historique :

    Seul le super utilisateur peut consulter cette page. Elle affiche la liste des derniers utilisateurs connectés, ainsi que la date et l'heure de leur connexion.

Gestion des groupes d'utilisateurs

  • Attribution de groupes :

    Si vous êtes autorisé à gérer les groupes d'utilisateurs, le bouton correspondant apparaîtra dans le menu. Cette page vous permet d'attribuer un utilisateur à un ou plusieurs groupes. Vous pouvez également supprimer un ou plusieurs groupes d'utilisateurs.

  • Vérification des groupes d'un utilisateur :

    Pour voir les groupes d'un utilisateur, sélectionnez d'abord l'utilisateur puis cliquez sur le bouton "Vérifier".

  • Attribution de groupes à un utilisateur :

    Sélectionnez d'abord l'utilisateur, puis les groupes que vous souhaitez lui attribuer. Cliquez ensuite sur le bouton "Attribuer".

Création et modification de groupes

  • Création d'un nouveau groupe :

    Si vous êtes autorisé à créer des groupes, le bouton correspondant apparaîtra dans le menu. Cette page vous permet de créer un nouveau groupe et de sélectionner les permissions que vous souhaitez lui attribuer.

  • Modification et suppression de groupes :

    Si vous êtes autorisé à modifier ou à supprimer des groupes, le bouton correspondant apparaîtra dans le menu.
